# Todo Manager
1. This is a project for testing purposses
## Keybings
### Save,load and quit
Everything starts from the Main Screen.
`l` - load

`s s` - save without quit

`s q` - try to save but then stop

`q s` - save and quit 

`q q` - quit without saving 

Space means press the first key, and then the second. 


### Navigation
`Arrows`

### Todos
`n` - new todo

`d` - delete todo

`e` - edit todo

### Todo editor
`Tab` for switching between text and value

`Arrows` for switching the type of the Todo

`Enter` while you are on the type to save the todo

`Escape` to close the popup
